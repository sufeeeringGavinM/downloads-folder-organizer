# downloads-folder-organizer
This thing organizes your downloads folder with Python!

# Installation 

0. Make sure you've got python installed!
1. Download the file downloads-organizer.py and put it in your downloads folder.

# Usage
1. Click downloads-organizer.py
2. Wait
3. It's done!

# Note
This makes a bunch of new folders named things like "APPLICATIONS", "ARCHIVES", "AUDIO", "CIRCUITS", etc etc... Change them up yourself if you want.
